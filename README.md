# 🦇 Wumpus World Logic Agent (AI Assignment Project)

## 📌 Project Overview
This project is a Web-based Dynamic Wumpus World Agent that uses a Knowledge-Based System (Propositional Logic) to navigate a grid environment. The agent perceives Breeze and Stench and uses inference rules to decide safe movement.

---

## 🚀 Features

- Dynamic grid size (user-defined)
- Random placement of:
  - Pits
  - Wumpus
- Knowledge Base (KB) updates in real time
- Basic logical inference (safe cell detection)
- Live percepts:
  - Breeze (Pit nearby)
  - Stench (Wumpus nearby)
- Game tracking:
  - Visited cells counter
  - Score system
  - Win / Game Over detection
- Clean Web UI (Flask + HTML/CSS/JS)

---

## 🧠 AI Concepts Used

- Knowledge Base (KB)
- Propositional Logic
- Rule-based reasoning
- Resolution idea (simplified)
- Percept-driven decision making
